# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of ShopVerse
- User authentication (register, login, profile management)
- Product browsing with search, filter, and sort
- Shopping cart functionality
- Guest checkout support
- Wishlist feature
- Product reviews (verified purchases only)
- Order tracking system
- Admin dashboard with analytics
- Product management (CRUD operations)
- Order management
- User management
- Responsive design with dark mode support
- Real-time updates using TanStack Query
- Smooth animations with Framer Motion

### Features

#### Shopping
- Browse products by category
- Search products by name
- Sort by price, rating, newest
- Add to cart (guest and authenticated users)
- Guest checkout with email
- Save shipping addresses (authenticated users)
- Multiple payment methods

#### User Account
- Secure authentication with JWT
- Profile management
- Order history
- Wishlist management
- Review purchased products
- Track orders in real-time

#### Admin Panel
- Dashboard with sales analytics
- Product management
- Order status updates
- User management
- Inventory tracking

#### UI/UX
- Modern, clean design
- Smooth page transitions
- Loading states and skeletons
- Toast notifications
- Mobile-responsive layout
- Dark mode support

### Technical
- React 18 with TypeScript
- Vite for fast development
- TanStack Query for data management
- Tailwind CSS for styling
- Framer Motion for animations
- Node.js/Express backend
- MongoDB database
- JWT authentication
- RESTful API design
